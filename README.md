# Excel_research_database

This project created a searchable database in Excel of 2500+ research papers from the Journal of Machine Learning Research (JMLR).





Data Dictionary:


    title: Paper title
    volume: Paper volume
    authors: Paper authors
    year: Released year
    pages: Number of pages
    link: Link to PDF
    code: Link to code

source of data:
https://www.jmlr.org
